# Productivity & Goal Tracking System

## 📌 Description

This is a web-based application to manage tasks and track goals. It is developed using Flask and follows Agile and DevOps practices.

## 🚀 Features

* User login system
* Add/Delete tasks
* Dashboard UI
* Data stored using SQLite database

## 🛠️ Tech Stack

* Frontend: HTML, CSS
* Backend: Python (Flask)
* Database: SQLite
* Version Control: GitHub

## 📂 Project Structure

* app.py → main backend
* templates → HTML pages
* static → CSS files
* database.db → database

## ⚙️ Run Project

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
